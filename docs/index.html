Steps to Make a Transaction

It takes only a few easy steps to make a transaction using Quik Slip

  • Purchase Slip Bookowners of Slip Books have the oportunity of making payments using Quik Slip
  • Scan QR codeThe owner of the slip book can scan the QR code on their payment slip to activate a payment slip.
  • Assign a valueAfter scanning, the payment slip can be given the value of the payment to be made.
  • Make paymentOnce the value is assigned, the payment can be made using the payment slip.
  • Redeeming SlipThe receiver can scan the QR code and retrieve the payment made.
  • Transaction completeThe transaction has now been completed.

Assignment of values to payment slips

The payee can easily assign a value to a payment slip using the app before making a payment. The process only requires scanning the QR code and assigning the required value and crossing if necessary.

Payment slip status

The user can easily check the status of the payment slip once a payment is made. Once a transaction is completed, both the payee and the reciever will be notified in real time.

Easy to make withdrawals

The reciever can withraw a payment by just scanning the QR code on the payment slip. The money will imediately be deposited to the reciever's account.

Accuracy

The transaction system is based on a Blockchain network. Hence the accuracy and speed are high. Auditing can also be done efficiently as all transactions will be logged on Blockchain. Any chance of fraud will be eliminated in the process.
